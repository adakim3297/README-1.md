## 2. 어떻게 사용하는 것인가?

* 코드 실행
  * 1. 바로 가기 사용Ctrl+Alt+N
  * 2. 또는 텍스트 편집기를 마우스 오른쪽 버튼으로 클릭한 다음 Run Code편집기 컨텍스트 메뉴에서 클릭
  * 3. 오른쪽 위 삼각형 클릭

* 실행중인 코드 중지
  * 1. 바로 가기 사용Ctrl+Alt+M
  * 2. 또는 Stop Code Run편집기 제목 메뉴에서 버튼 클릭
